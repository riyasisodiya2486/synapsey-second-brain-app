🧠Synapsey (Second Brain App)

A personal AI-powered vault for saving and querying content.
Drop in links from YouTube, Instagram, Twitter/X, and more, then ask natural language questions to instantly find what matters.
________________________________________

✨ What You Can Do (For Users)
•	🔗 Save Links – Add YouTube, Instagram, and Twitter/X posts into your vault.
•	🧠 Ask Anything – Use AI to query across your saved links like a personal knowledge base.
•	📂 Stay Organized – All your resources are stored in one place, searchable and retrievable.
•	⚡ Instant Retrieval – Forget scrolling through bookmarks – just ask.
•	🔒 Your Data, Your Vault – Private and secure.
________________________________________

🎯 How to Use
1.	Add a link → Paste a YouTube, Instagram, or Twitter/X link into your vault.
2.	Save it → The app stores and indexes the content.
3.	Ask a question → Query your vault in plain English.
4.	Get answers → AI summarizes and retrieves the most relevant info.
Example:
“What were the 3 key points from that YouTube video I saved about startups?”
________________________________________

🚀 Getting Started (For Developers)
1️⃣ Clone the repo
    git clone https://github.com/riyasisodiya2486 /synapsey
    cd synapsey
2️⃣ Install dependencies    
    npm install
3️⃣ Setup environment variables
    Create a .env file in the root directory:
    PORT=5000
    DATABASE_URL=your_database_url
    OPENAI_API_KEY=your_openai_api_key
4️⃣ Run the development server
    npm run dev
App will be live at: http://localhost:5000 🎉
